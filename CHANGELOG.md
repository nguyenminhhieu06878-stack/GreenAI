# Changelog

All notable changes to GreenEnergy AI will be documented in this file.

## [1.0.0] - 2024-01-01

### 🎉 Initial Release

#### ✨ Features

**Frontend**
- Dashboard với biểu đồ tiêu thụ điện trực quan
- Authentication (Login/Register)
- Nhập chỉ số điện qua chụp ảnh hoặc thủ công
- Biểu đồ phân tích tiêu thụ (Line, Bar, Pie charts)
- Trang mẹo tiết kiệm điện
- Bảng giá dịch vụ
- Responsive design với TailwindCSS
- State management với Zustand
- Data fetching với React Query

**Backend**
- RESTful API với Express + TypeScript
- JWT Authentication
- Upload và xử lý ảnh đồng hồ điện
- API endpoints cho analytics
- Error handling middleware
- CORS configuration

#### 🛠️ Technical Stack

- React 19
- TypeScript 5.3
- Vite 5
- TailwindCSS 3.4
- Node.js + Express
- JWT + Bcrypt
- Recharts
- Zustand
- React Query

- CONTRIBUTING.md - Hướng dẫn đóng góp
