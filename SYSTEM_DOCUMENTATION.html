📡 API Endpoints

Authentication

  • POST /api/auth/register - Đăng ký tài khoản mới
  • POST /api/auth/login - Đăng nhập email/password
  • GET /api/auth/google - Đăng nhập Google OAuth
  • GET /api/auth/google/callback - Google OAuth callback
  • GET /api/auth/me - Lấy thông tin user hiện tại

Meter Readings

  • GET /api/meter/readings - Lấy danh sách chỉ số
  • POST /api/meter/readings - Thêm chỉ số mới
  • POST /api/meter/ocr - Upload ảnh và OCR

Rooms

  • GET /api/rooms - Danh sách phòng
  • POST /api/rooms - Tạo phòng mới
  • GET /api/rooms/:id - Chi tiết phòng
  • PUT /api/rooms/:id - Cập nhật phòng
  • DELETE /api/rooms/:id - Xóa phòng

Payments

  • GET /api/payment/pending - Hóa đơn chờ thanh toán
  • POST /api/payment/create - Tạo link thanh toán PayOS
  • POST /api/payment/webhook - PayOS webhook callback

AI Services

  • POST /api/ai/chat - Chat với AI chatbot
  • POST /api/ai/tips - Nhận gợi ý tiết kiệm điện

Admin

  • GET /api/admin/users - Quản lý users
  • GET /api/admin/stats - Thống kê hệ thống
  • GET /api/admin/revenue - Báo cáo doanh thu

🔄 Workflow & User Journey

Tenant Journey

  1. Đăng ký/Đăng nhập: Email hoặc Google OAuth
  2. Chọn vai trò: Tenant (người thuê)
  3. Được assign phòng: Landlord gán phòng cho tenant
  4. Xem dashboard: Theo dõi tiêu thụ điện nước
  5. Chụp đồng hồ: Upload ảnh, OCR tự động đọc số
  6. Nhận hóa đơn: Hệ thống tính toán tự động
  7. Thanh toán: PayOS online payment
  8. Check-in hàng ngày: Nhận điểm thưởng
  9. Nhận gợi ý AI: Tips tiết kiệm điện

Landlord Journey

  1. Đăng ký/Đăng nhập: Email hoặc Google OAuth
  2. Chọn vai trò: Landlord (chủ nhà)
  3. Chọn gói subscription: Cơ bản/Pro/Enterprise
  4. Tạo phòng trọ: Thêm thông tin phòng
  5. Thêm tenant: Gán người thuê vào phòng
  6. Nhập chỉ số: Cập nhật số điện nước hàng tháng
  7. Tạo hóa đơn: Hệ thống tự động tính tiền
  8. Theo dõi thanh toán: Xem trạng thái hóa đơn
  9. Xem báo cáo: Analytics và doanh thu

🛡️ Security Best Practices

  • ✅ Password hashing với bcrypt (10 rounds)
  • ✅ JWT tokens với expiration time
  • ✅ CORS configuration cho production domains
  • ✅ Environment variables cho sensitive data
  • ✅ Input validation với Zod
  • ✅ MongoDB injection prevention
  • ✅ Rate limiting (có thể thêm)
  • ✅ HTTPS cho tất cả connections

📱 Responsive Design

  • Mobile-first approach với Tailwind CSS
  • Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
  • Touch-friendly UI cho mobile devices
  • Optimized cho cả desktop và mobile

⚠️ Known Issues & Limitations

Google OAuth:
• Cần cấu hình đúng redirect URI trong Google Cloud Console
• Callback URL phải khớp chính xác với Railway domain
• Có thể mất 5-10 phút để Google cập nhật cấu hình mới
OCR Accuracy:
• Độ chính xác phụ thuộc vào chất lượng ảnh
• Cần ảnh rõ nét, đủ sáng
• Đồng hồ điện phải hiển thị rõ ràng
Real-time Chat:
• Hiện tại dùng HTTP polling thay vì WebSocket
• Vercel không hỗ trợ WebSocket tốt
• Có thể nâng cấp lên Socket.io với Railway
